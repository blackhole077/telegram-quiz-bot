## Grading rubric

Score each problem from 0.0 to 1.0 using the tiers below. Apply the same criteria to every problem.

### Partial credit tiers

- 1.0 - Fully correct: correct method, correct working, correct final answer (or equivalent)
- 0.8 - Method correct, minor error: correct approach and working, but a small arithmetic or notation mistake leads to a wrong final answer
- 0.6 - Method correct, significant gap: correct approach identified but a key step is missing or a conceptual error limits the working
- 0.4 - Partially correct: correct identification of the relevant concept or formula, but the method is not fully applied
- 0.2 - Minimal credit: evidence the student understood what was being asked, but the approach is wrong
- 0.0 - No credit: blank, off-topic, or demonstrates a fundamental misunderstanding of the question

### Multi-part problems

Score each sub-part using the same tiers and report the average for the problem score.

### Common cases

- Correct final answer with no working shown: award 0.5 at most (method unverifiable)
- Correct method, wrong intermediate result propagated correctly through to final answer: award 0.8
- Copied formula without applying it: award 0.2

### Scoring rule

Each problem score = one tier value from the list above
total_score = average of all problem scores (equal weight unless the problem specifies otherwise)

### Consistency guidance

- Penalise missing working only once - do not also penalise for the downstream errors it causes
- A student who identifies the right approach but makes an early error and carries it through correctly deserves at least 0.6 (not 0.0)
- Grade what the student demonstrated, not what you infer they meant to write
- When a problem asks for explanation and the student only provides a formula, treat it as missing working
