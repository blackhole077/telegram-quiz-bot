## Grading rubric

Score from 0.0 to 1.0. The explanation is evaluated on four axes, each with a maximum of 0.25.

### Axis 1 - Correctness (max 0.25)

Are all factual claims in the explanation accurate?

- 0.25: No significant errors; any simplifications are appropriate for the audience
- 0.15: One minor factual error that does not undermine the main point
- 0.05: One significant factual error, or several minor ones
- 0.0: The explanation is fundamentally incorrect or contains a central misconception

### Axis 2 - Completeness (max 0.25)

Does the explanation cover the key ideas expected for the stated audience?

- 0.25: All essential points for the audience level are present
- 0.15: One important point is missing but the explanation is otherwise adequate
- 0.05: Multiple important points are absent
- 0.0: So incomplete it would leave the audience without a functional understanding

### Axis 3 - Audience fit (max 0.25)

Is the vocabulary, depth, and framing calibrated to the stated audience?

- 0.25: Well-calibrated: right vocabulary level, appropriate depth, no unexplained jargon
- 0.15: Mostly appropriate with one minor miscalibration
- 0.05: Noticeable mismatch - too technical or too simplified for the stated audience
- 0.0: Clearly written for a completely different audience

### Axis 4 - Analogy quality (max 0.25)

If an analogy is used, does it illuminate the concept accurately without misleading?

- 0.25: Analogy is structurally sound and illuminates the key mechanism
- 0.15: Analogy is helpful but slightly misleading on a secondary point
- 0.05: Analogy is vivid but structurally misleading in a way that could cause misconceptions
- 0.0: Analogy actively contradicts the concept
- 0.2: No analogy used AND the explanation is otherwise clear (neutral - not penalised, not rewarded)

### Scoring rule

total score = axis_1 + axis_2 + axis_3 + axis_4
correct = true if total >= 0.7

### Consistency guidance

- A vivid but structurally wrong analogy is worth less than a plain but accurate explanation
- Completeness is judged relative to the audience: beginners need core intuition, experts need edge cases
- Never award axis_1 >= 0.15 if a central misconception is present, even if the rest is polished
- When no analogy is used and none was needed, award axis_4 = 0.2 (neutral)
